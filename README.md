
# 🦠 EpiTrack — COVID-19 Epidemic Dashboard

Live Dashboard: (add after deploy)

## About
AI-powered epidemic spread prediction dashboard built for Codecure AI Hackathon at IIT BHU.

## Features
- Real-time COVID-19 trend analysis
- Global risk map
- 30-day outbreak forecast
- Country comparison

## Tech Stack
Python, Streamlit, Plotly, Pandas, Scikit-learn

## Data Source
Johns Hopkins CSSE COVID-19 Dataset

## Run Locally
pip install -r requirements.txt
streamlit run app.py
